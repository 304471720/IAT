# 平台简介
VFT(Victorinox For Test)是一个及多平台自动化于一体的web版的自动化测试平台。

Web自动化底层通过RobotFramework执行用例，API自动化底层通过Jmeter执行用例。前端通过react js开发，后端通过Flask提供接口服务，数据入库至Mysql。

包含了人员控制，团队协作，版本控制、定时任务等测试相关常用功能。

- [x] Web UI自动化
- [x] API 接口自动化
- [ ] APP UI自动化（待开发）
